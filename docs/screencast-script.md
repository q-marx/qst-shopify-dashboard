# Screencast Script

1. Start in Shopify Admin and open QST Listing Workspace.
2. Show the Overview with the embedded state and Shopify shop context.
3. Open Products and demonstrate search, filters, and product selection.
4. Open Listing review for one selected product and show title, description, image, price, variants, tags, product type, and vendor.
5. Open eBay connection and show the environment label.
6. Connect eBay using the sandbox/review account, or show the already-connected OAuth state.
7. Save prepared listing records for the selected products.
8. Show validation success or specific validation issues.
9. Download an eBay batch/export pack and show the recent export activity.
10. Show the Desktop companion card and state that desktop is optional for advanced desktop-first workflows.

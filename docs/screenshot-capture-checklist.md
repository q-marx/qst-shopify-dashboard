# Screenshot Capture Checklist

Capture five real UI screenshots from the embedded Shopify Admin app:

1. Overview dashboard with genuine Shopify and eBay connection states.
2. Product browser with filters and batch selection.
3. Listing review with real product data, variants, images, and validation checks.
4. eBay connection or listing preparation flow with the environment label visible.
5. Export pack or prepared-listing/result activity screen.

Do not use promotional collages, browser chrome, desktop wallpaper, fake metrics, Partner Dashboard forms, or mocked connected states.

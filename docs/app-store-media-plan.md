# App Store Media Plan

Last updated: 2026-07-02

## Where Screenshots Go

Upload final screenshots in Shopify Partner Dashboard:

1. Apps > All Apps
2. QST Listing Workspace
3. Distribution
4. Shopify App Store listing > Manage listing
5. Edit the published language or complete the listing content
6. Add screenshots under the listing media/screenshots section

Keep local working copies in this repo under:

`docs/app-store-media/`

or in the existing promo folder:

`..\promo\`

Do not rely on screenshots committed to the repo as the submission source. The actual App Store submission uses the Partner Dashboard uploads.

## Capture Rules

- Capture inside Shopify Admin, not the plain Render URL.
- Use real dev-store products.
- Avoid browser chrome, Partner Dashboard, Render Dashboard, secrets, tokens, and private emails.
- Do not show a required eBay connection step.
- Do not show unsupported claims such as live marketplace publishing from the web dashboard.
- Confirm no text is clipped at desktop and mobile widths before using screenshots.

## Feature Media

Shopify recommends feature media that helps merchants understand the app. If using a video, keep it short and focused on the value rather than a long tutorial.

Recommended video flow:

1. Open QST Listing Workspace in Shopify Admin.
2. Show products loaded from Shopify.
3. Filter/search products.
4. Open one listing draft.
5. Edit local draft text.
6. Mark export setup notes.
7. Download an eBay CSV pack or listing pack.
8. Show QST Desktop as optional.

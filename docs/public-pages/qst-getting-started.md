# Getting Started With QST Listing Workspace

Publication note: review and adapt before publishing on Q-MER.CH.

## What QST Listing Workspace Is

QST Listing Workspace is a Shopify Admin app for preparing marketplace export packs from Shopify product data.

It is read-only for Shopify products. QST does not change your Shopify catalogue.

## Step 1: Open QST In Shopify Admin

Open Shopify Admin, then Apps, then QST Listing Workspace.

Product loading works inside Shopify Admin because QST uses Shopify's embedded app context. Opening the public Render URL directly will not show live Shopify products.

## Step 2: Review The Overview

The overview shows:

- products loaded
- products ready to export
- products needing review
- visible variants
- recent prepared records and exports
- optional QST Desktop pairing status

## Step 3: Find Products

Use search and filters to find products by title, SKU, tag, product type, Shopify status, readiness, or local work status.

## Step 4: Review A Listing Draft

Select a product to review:

- marketplace title
- description
- suggested tags
- price and SKU
- images
- variants
- readiness checks

Draft edits are local to QST. They do not update Shopify.

## Step 5: Use Export Setup Notes

The export setup notes are optional review markers for details that are not always present in Shopify product data:

- Policy notes added: payment, return, fulfilment, or marketplace seller rules are captured.
- Dispatch location added: country, postcode, or stock location is captured.
- Fallback category note added: a category or search hint is captured for products that need category review.

These notes help make exports easier to review before importing elsewhere or continuing in QST Desktop.

## Step 6: Export

Select one or more products, then download:

- CSV
- listing pack
- workspace pack
- eBay-compatible CSV pack
- review plan

Exports are generated from Shopify product data and local QST draft choices.

## Optional: Pair QST Desktop

QST Desktop is optional. Merchants who want advanced local workflows can download the Windows companion, generate a pairing code in Shopify Admin, and enter it in QST Desktop.

The Shopify app's core workflow works without QST Desktop.

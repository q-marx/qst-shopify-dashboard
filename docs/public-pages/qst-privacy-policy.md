# QST Privacy Policy Draft

Publication note: this is product-specific draft copy for QST. Review with legal advice before publishing.

## Overview

QST Listing Workspace helps Shopify merchants prepare marketplace listing drafts and export packs from Shopify product data. QST uses Shopify product information in read-only mode and does not write changes back to Shopify products.

## Information QST Processes

QST may process:

- Shopify shop domain and app installation context.
- Shopify product data made available through the app's `read_products` scope, such as titles, descriptions, handles, vendors, product types, tags, statuses, images, variants, prices, SKUs, and inventory quantities.
- Listing draft and export-preparation records created by the merchant.
- Export activity records, such as export type, product count, filenames, and timestamps.
- Optional desktop pairing codes and desktop pairing tokens.
- Optional eBay OAuth tokens if the merchant connects eBay in QST Desktop.
- Technical logs needed to operate, secure, and troubleshoot the app.

QST does not request Shopify customer, order, payment, fulfilment, or write-product scopes for the Shopify Admin app.

## How QST Uses Information

QST uses information to:

- load products in the embedded Shopify Admin app
- prepare listing drafts and export packs
- save prepared listing records for the installed shop
- support optional QST Desktop pairing
- diagnose errors and secure the service
- respond to merchant support requests

QST does not sell merchant data.

## Storage

The Shopify Admin app stores server-side records in the production application database used by QST. Browser-local draft edits and image choices may also be stored in the merchant's browser local storage so work survives a refresh.

QST Desktop stores local preferences, tokens, caches, diagnostics, and export files on the merchant's Windows device.

## Third Parties

QST uses Shopify to authenticate merchants and read Shopify product data. QST may use hosting and database providers to operate the app. If a merchant chooses to use QST Desktop eBay publishing, eBay authentication and marketplace APIs are used for that desktop workflow.

## Retention And Deletion

QST keeps operational records for as long as needed to provide the app, maintain security, troubleshoot issues, and meet legal obligations. When a merchant uninstalls the Shopify app or requests deletion, QST will process the request according to Shopify's required privacy webhook flow and applicable law.

## Merchant Choices

Merchants can:

- uninstall the Shopify app from Shopify Admin
- clear browser-local draft data from the QST dashboard
- disconnect optional desktop marketplace accounts inside QST Desktop
- contact Q-MER.CH for support or deletion requests

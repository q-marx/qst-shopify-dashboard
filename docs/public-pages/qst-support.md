## QST Listing Workspace Support

QST Listing Workspace is a Shopify Admin app for preparing marketplace export packs from Shopify product data. It uses read-only Shopify product access and does not change your Shopify catalogue.

## What QST Does

- Loads Shopify product information inside Shopify Admin.
- Lets you search, filter, and review products.
- Helps prepare local listing drafts.
- Saves prepared listing records for review.
- Downloads CSV, listing pack, workspace pack, and review plan exports.
- Offers QST Desktop as an optional Windows companion for advanced local workflows.

## What QST Does Not Do In The Shopify App

- It does not edit Shopify products.
- It does not require QST Desktop.
- It does not require an eBay account connection for the Shopify Admin workflow.
- It does not automatically publish live marketplace listings from the web dashboard.

## Basic Workflow

1. Open QST Listing Workspace from Shopify Admin.
2. Review the product metrics and product list.
3. Search or filter the products you want to prepare.
4. Select a product and review the listing draft.
5. Edit the draft locally if needed.
6. Select products for export.
7. Save prepared records or download export packs.

## Export Setup Notes

The export setup checkboxes are optional review markers:

- Policy notes added: payment, return, fulfilment, or marketplace seller rules have been captured.
- Dispatch location added: a country, postcode, or stock location has been recorded for marketplace review.
- Fallback category note added: a default category or search hint has been recorded for products needing category review.

## Troubleshooting

If products do not load, open QST from inside Shopify Admin rather than from the plain Render URL. Product loading requires Shopify Admin context.

If the app asks you to re-authorise Shopify, use the re-authorise button in the dashboard and return to QST after Shopify approves access.

If an export button is disabled, select at least one product or use the readiness filters to find export-ready products.

If QST Desktop pairing is used, generate the code in QST Listing Workspace and enter it in QST Desktop before the code expires.
